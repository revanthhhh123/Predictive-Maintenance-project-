# Predictive Maintenance Using Machine Learning

## Project Overview

This project predicts whether an industrial machine is likely to fail based on operational parameters. A synthetic dataset is generated and analyzed using Machine Learning techniques. The project includes data preprocessing, exploratory data analysis (EDA), feature scaling, model training, performance evaluation, feature importance analysis, and sample prediction.

The goal is to demonstrate how machine learning can assist industries in identifying potential machine failures before they occur, thereby reducing maintenance costs and minimizing unexpected downtime.

## Features

* Synthetic dataset generation
* Data preprocessing and label encoding
* Exploratory Data Analysis (EDA)
* Correlation heatmap visualization
* Machine failure distribution analysis
* Feature scaling using StandardScaler
* Logistic Regression model
* Decision Tree model
* Random Forest model
* Confusion matrix visualization
* Classification report
* Model accuracy comparison
* Feature importance analysis
* Sample prediction
* Trained model saved using Joblib

## Installation

1. Clone the repository

```bash
git clone https://github.com/revanthhhh123/Predictive-Maintenance-project-.git
```

2. Navigate to the project folder

```bash
cd Predictive-Maintenance-project-
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn joblib
```

4. Run the project

```bash
python project.py
```
